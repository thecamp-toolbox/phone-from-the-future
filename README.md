# phone-from-the-future
A device that allows you to listen to conversations from the future.

- Arduino UNO 
- Arduino MP3 shield : https://www.adafruit.com/product/1790
- micro SD card 
- keypad : https://www.adafruit.com/product/1824

- small speakers
